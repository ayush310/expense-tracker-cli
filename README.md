# Expense Tracker CLI

A command-line expense tracker built to practice Python fundamentals —
classes, dataclasses, validation, JSON persistence, and pytest.

## Features

- Add, filter, and delete expenses
- Category-wise totals
- JSON-based persistence across runs
- Input validation (amount, category, date format)

## Run it

```bash
python expense.py
```

## Run tests

```bash
pytest -v
```

## Stack

Python 3.12 · dataclasses · pathlib · pytest
